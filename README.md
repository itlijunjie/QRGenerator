# QRGenerator
    mac 下用来生成二维码的命令行工具
## -outpath和-content
    -outpath 输出文件的路径
    -content 二维码存放的内容
## 举例
    ./QRGenerator -outpath ~/Desktop/test.png -content http://www.baidu.com

暂时不支持size的设置默认大小是300*300
